# 📁 文件处理器 Pro

一个功能强大的文件批量处理工具，提供直观的图形界面，帮助您高效管理文件和文件夹。

## ✨ 功能特性

### 📂 文件操作
- **获取名称** - 递归扫描目录，导出所有文件和文件夹信息到 Excel（包含类型、大小、修改时间等）
- **修改名称** - 根据 Excel 表格批量重命名文件
- **修改后缀** - 根据 Excel 表格批量修改文件扩展名
- **创建文件/文件夹** - 根据 Excel 表格批量创建文件和目录结构
- **删除文件** - 根据 Excel 表格中的删除标记批量删除文件

### 💾 文件备份
- **增量同步** - 只同步新增或修改的文件
- **单向同步** - 将源目录完全同步到目标目录
- **镜像同步** - 双向同步，保持两个目录完全一致
- **备份组管理** - 保存和加载备份配置，一键执行备份任务

### ⚡ 批量处理
- **一键处理模板** - 生成包含所有操作列的 Excel 模板
- **模板创建** - 快速创建标准格式的 Excel 处理文件

## 📋 Excel 模板格式

| 列 | 字段 | 说明 |
|---|---|---|
| A | 源文件名/路径 | 要操作的文件或文件夹名称（支持相对路径） |
| B | 新文件名 | 重命名时的目标名称 |
| C | 新后缀 | 修改后缀时的目标扩展名 |
| D | 是否删除 | 填写 `delete` / `是` / `yes` 标记删除 |

## 🚀 快速开始

### 环境要求
- Windows 10 或更高版本
- Python 3.8+

### 安装依赖

```bash
pip install PyQt5 pandas dirsync requests openpyxl
```

### 运行程序

```bash
python "文件处理器 pro.py"
```

### 打包为可执行文件

```bash
pyinstaller --onefile --noconsole --icon=logo.ico --name=文件处理器 "文件处理器 pro.py"
```

## 📖 使用指南

### 1. 获取文件列表

1. 点击「浏览」选择目标文件夹
2. 点击「获取名称」按钮
3. 选择保存位置，自动生成包含所有文件信息的 Excel

### 2. 批量重命名

1. 准备好 Excel 文件：
   - 第1列：当前文件名
   - 第2列：新文件名
2. 选择文件夹
3. 点击「修改名称」并选择 Excel 文件

### 3. 批量修改后缀

1. 准备好 Excel 文件：
   - 第1列：当前文件名
   - 第2列：新后缀（如 `txt`、`pdf`）
2. 选择文件夹
3. 点击「修改后缀」并选择 Excel 文件

### 4. 批量创建

1. 准备好 Excel 文件，第1列填写要创建的路径
2. 文件名包含点号（如 `test.txt`）创建文件
3. 文件名不含点号创建文件夹
4. 支持多级目录（如 `folder/subfolder/file.txt`）

### 5. 批量删除

1. 准备好 Excel 文件：
   - 第1列：要删除的文件/文件夹名
   - 第2列：填写 `delete` 标记
2. 选择文件夹
3. 点击「删除文件」并选择 Excel 文件

### 6. 文件备份

1. 在备份标签页设置源路径和目标路径
2. 选择同步模式
3. 可选：保存为备份组方便下次使用
4. 点击「开始备份」

### 7. 一键处理

1. 点击「一键处理」生成模板 Excel
2. 编辑 Excel，填写需要操作的内容
3. 使用各个功能按钮分别执行操作
