# File-processor

一个基于 PyQt5 开发的图形化文件管理工具，提供文件/文件夹的批量重命名、后缀修改、创建/删除、备份同步等功能。

## 功能特性

### 基础文件操作
- **路径选择** - 支持选择文件或文件夹作为工作目录
- **获取名称** - 递归列出目录下所有文件和文件夹，按类型用不同颜色标识：
  - 蓝色：目录/文件夹
  - 品红：图片文件
  - 石板灰：文本/文档文件
  - 红色：压缩文件
  - 绿色：可执行文件
  - 粉色：音频文件
  - 橙色：视频文件
- **批量重命名** - 通过 Excel 文件批量修改文件名（第一列为原文件名，第二列为新文件名）
- **批量修改后缀** - 批量修改指定后缀的文件扩展名
- **创建文件/文件夹** - 根据 TXT 文件的缩进结构，自动创建文件夹和文件
- **删除文件** - 待完善
- **备份文件** - 支持多种同步模式的文件备份

### 备份同步功能
- **增量同步** - 仅同步源目录中新增或修改的文件
- **单向同步** - 将源目录内容同步到目标目录，删除目标目录中多余的文件
- **镜像同步** - 双向同步，保持两个目录完全一致
- **备份组管理** - 保存多个备份配置，支持快速加载、删除

### 其他功能
- **日志输出** - 实时显示操作日志，支持保存日志到文件
- **检查更新** - 自动检测 GitHub 上的新版本

## 安装与运行

### 环境要求
- Python 3.6+
- 依赖库：`PyQt5`, `pandas`, `openpyxl`, `requests`, `dirsync`

### 安装步骤

1. 克隆或下载项目代码
2. 安装依赖：
```bash
pip install PyQt5 pandas openpyxl requests dirsync
```
3. 运行程序：
```bash
python 文件处理器.py
```

## 使用说明

### 基本操作流程
1. 点击「选择」按钮，选择要操作的文件夹或文件
2. 根据需要点击对应的功能按钮

### 批量重命名（Excel 方式）
1. 准备一个 Excel 文件（.xlsx 或 .xls）
2. 第一列填写当前文件名，第二列填写目标文件名
3. 点击「修改名称」按钮，选择该 Excel 文件
4. 程序自动执行重命名，并在日志区显示结果

### 批量修改后缀
1. 选择目标文件夹
2. 点击「修改后缀」按钮
3. 在弹出的对话框中：
   - 输入或选择「源后缀」（如 `txt`）
   - 输入或选择「修改为」的目标后缀（如 `log`）
4. 点击「确定」执行

### 根据 TXT 文件创建结构
TXT 文件格式示例：
```
项目目录
    子文件夹1
        文件1.txt
        文件2.log
    子文件夹2
        图片.jpg
    根目录文件.md
```
- 带扩展名的行会被创建为**文件**
- 不带扩展名的行会被创建为**文件夹**
- 缩进（4个空格）表示层级关系

### 备份同步
1. 点击「备份文件」按钮打开备份窗口
2. 选择「源路径」和「目标路径」
3. 选择同步模式（增量/单向/镜像）
4. 可选：点击「保存」将配置保存为备份组
5. 点击「开始」执行备份
